Apify actors do work today, but expect occasional 429s/blocks. Acceptable for a personal app — flag this when sharing publicly.\n- **Latency cost**: live mode adds **15–35 seconds** to every generation (Apify cold start 5–10s + scrape 5–15s + 6–8 vision calls 5–10s). I'll mitigate with parallel vision calls and a 24-hour cache so repeat (occasion+mood+archetype) combos are instant.\n- **$ cost**: ~$0.01–0.03 Apify + ~$0.005 Lovable AI vision per generation.
